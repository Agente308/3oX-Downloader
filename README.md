# 3oX-Downloader
Este programa te permite descargar videos con distintos formatos y calidades de  plataformas como Facebook, TikTok, etc ademas sin marca de agua.

Para que este funcione necesitas intalar las siguientes librerias de python

• yt-dlp

• PyQt5

• certifi

Luego de eso instala FFmpeg en tu sistema con el siguiente tutorial

• https://www.youtube.com/watch?v=JR36oH35Fgg

Ahora que tienes todo esto ejecuta el programa y en configuracion ingresas la ruta donde esta ffmpeg.exe
